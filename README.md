# Smart Advisor: AI-Powered Academic Guidance System

**Smart Advisor** — это высокотехнологичная корпоративная платформа для прецизионного управления образовательными траекториями. Система объединяет классические алгоритмы скоринга с современными агентными архитектурами (LLM Agents) и векторным поиском (RAG) для создания бесшовного опыта профессионального развития.

## 🚀 Основные возможности (Features)

*   **Интеллектуальный подбор курсов**: Многофакторная система гибридного скоринга, учитывающая разрыв в навыках (Skill Gap), семантическую близость контента и персональные предпочтения.
*   **AI Academic Advisor**: Интерактивный агент на базе архитектуры ReAct, способный вести диалог, искать дополнительные материалы (Coursera, edX, YouTube) через Tavily API и давать карьерные советы.
*   **Генератор планов обучения**: Автоматическое создание пошаговых образовательных траекторий (Learning Plans) с четкими этапами достижения целей.
*   **Визуализация Skill Graph**: Интерактивное отслеживание прогресса и визуализация графа навыков студента в реальном времени.
*   **RAG-анализ материалов**: Глубокий анализ силлабусов и учебных заметок с использованием векторного поиска (PGVector) для повышения точности рекомендаций.
*   **Real-time Dashboard**: Централизованная панель управления для мониторинга академической активности и прогресса по планам.
*   **Локализация (i18n)**: Полная поддержка английского и русского языков "из коробки".

## 🏗 Техническая архитектура

Система построена на принципах Clean Architecture и событийно-ориентированного взаимодействия.

### Стек технологий (Tech Stack)
*   **Backend**: Python 3.11+, FastAPI, SQLAlchemy 2.0, Pydantic v2, Alembic.
*   **AI/ML Engine**: LlamaIndex, OpenAI GPT-4o/5, PGVector, Tavily Search API.
*   **Frontend**: Next.js 16 (App Router), React 19, TailwindCSS 4, TanStack Query, Zustand.
*   **Infrastructure**: Docker & Docker Compose, Redis (Arq for background tasks), PostgreSQL.

### Потоки данных (Data Flow)
1.  **Ingress**: Запросы от фронтенда поступают на FastAPI Gateway.
2.  **Orchestration**: Тяжелые AI-задачи (генерация планов, глубокий анализ) делегируются в очередь **Redis/Arq**.
3.  **Execution**: Фоновые воркеры взаимодействуют с LLM и векторным индексом.
4.  **Persistence**: Результаты сохраняются в PostgreSQL, а клиент получает обновление через механизм синхронизации состояния.

## 🧠 Логика принятия решений (AI Recommendation Flow)

Процесс формирования рекомендаций проходит через три критические фазы:

### 1. Гибридный скоринг (Scoring Engine)
Финальный балл курса вычисляется по формуле:
`Score = (Sim_content * 0.3) + (Gap_skill * 0.3) + (Score_RAG * 0.2) + (Score_pref * 0.2)`

*   **Skill Gap**: Оценка объема *новых* навыков, которые получит студент относительно текущего профиля.
*   **Semantic Search**: Поиск по векторным эмбеддингам описаний курсов через `pgvector`.

### 2. Агентное обоснование (Agentic Reasoning)
**ReAct Agent** анализирует отобранные курсы, используя инструменты:
*   **search_external_resources**: Поиск на Coursera/edX, если внутренние курсы не закрывают потребность.
*   **RAG Query**: Извлечение деталей из силлабусов и учебных материалов.

### 3. Структурированный вывод
Агент формирует финальный вердикт в формате JSON, проходящий через строгую валидацию **Pydantic/Zod**, что гарантирует консистентность данных в UI.

## 🛠 Установка и запуск (Installation & Usage)

### Предварительные требования
*   Docker & Docker Compose
*   OpenAI API Key
*   Tavily API Key

### Быстрый старт
1.  Клонируйте репозиторий.
2.  Создайте `.env` файл на основе `.env.example` и заполните API ключи.
3.  Запустите систему:
    ```bash
    docker-compose up --build
    ```
4.  Откройте `http://localhost:2929` для доступа к интерфейсу.
